# Clothing Shop

## Описание
Простой интернет-магазин одежды на HTML, CSS и JavaScript.
Проект является пет-проектом и используется для практики фронтенд-разработки.

## Функционал
- Просмотр Главной страницы
- Просмотр каталога товаров
- Фильтрация товаров
- Корзина покупок
- Страницы товара, контакты


## Технологии
- HTML5
- CSS3
- JavaScript (vanilla)
- Git + GitHub Pages

## Запуск проекта локально
1. Клонировать репозиторий:  
   `git clone https://github.com/Z4khar/Clothing-shop.git`
2. Открыть `index.html` в браузере  
   или запустить локальный сервер (например, через Live Server в VSCode/WebStorm)
